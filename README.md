## Features

- Fully responsive layout
- Multilanguage support (EN / ES) using i18n
- Clean UI focused on readability
- Sections for projects, experience, and contact information

## Tech Stack

- React + TypeScript
- CSS
- Vite
- i18next (internationalization)
- Deployed with Vercel

&nbsp;
## How to run locally

### Pre-requisites
- Node.js (v22)
- npm

### Installation
```bash
git clone https://github.com/felipe-u/portfolio.git
cd portfolio
npm install
```

### Run
```bash
npm run dev
```
